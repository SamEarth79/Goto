# GoTo Bengaluru

A simple web app to discover places to visit in Bengaluru.

## Idea

A browsable site that lists interesting places around Bengaluru — parks, cafes,
viewpoints, monuments, breweries, etc. — with basic details for each (name,
area/locality, description, photo).

## Status

Planning stage — no code yet. Working through the following before building:

- [ ] **Scope** — which categories of places to include
- [ ] **Data source** — hand-curated list vs. external API (Google Places, OSM)
- [ ] **Features** — list/grid view, filtering by category/area, search, map view
- [ ] **Tech stack** — static site vs. small framework (e.g., React/Next.js),
      deployment target (GitHub Pages, Vercel, local only)
- [ ] **Level of "simple"** — basic list with info cards vs. more interactive
      (maps, ratings, reviews)

## Next steps

Finalize the above decisions, then sketch the data model and page structure.
